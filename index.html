Y69 Electric Strike Set

The Y69 Electric Strike Set is designed for security gates and internal doors. Available in two configurations:

  • Plug-In Type — Single pulse, adjustable latch, 12V AC transformer included. Suitable for steel security gates and wooden doors.
  • Bell Type — Auto-lock on door close, manual key override, 2-year warranty. Ideal for home and business.

Custom branding & packaging available (OEM/ODM).
